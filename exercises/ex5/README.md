# Analyse  

In this part if the exercise, you are taking the persona of a Integration Manager. As an Integration Manager, you ould want to see how integration pattern, styles etc is used within your organization and have the ask to analyse data. This section would allow you to to that. In this exercise, we are just going to understand various options.

## Steps for Analysis

Run through the following steps from the Integration Assesement Tool Landing Page:

1. From main landing page, Click **Analyse**
<br><img src="/exercises/ex5/images/1.1_Ex4_Create_Request_Main_Page.PNG" width=80% height=80%>

2. Click **Intergartion Areas**
<br><img src="/exercises/ex5/images/1.2_Ex4_Select_Interface_Request.PNG" width=80% height=80%>

3. Details of all Integration Areas are displayed. This is displayed as a table of Integration Styles and Integration Domains.



4. Select the first Coud2OnPremise Green Box. For instructor, this is IP-2
    - Click **Related Technologies**
    - Displays all the Related Technologies configured
    - Click < to go back



5. Click **Related Interface Requests**
    - Displays related interface requests.
    - Click < to go back
    - Click x to Exit




6. Click **Integration Policies**
    - Review various Integration Policies. 



7. Click **Application Overview**.
    - The application overview depicts the existing connection between a source and a target application within the IT landscape.



8. Click **Integration Technologies Overview**
    - Integration technologies describe the capability coverage and usage of middleware components. 
    - Using integration technologies, you can customize existing technology profiles or create new integration technology as per your organizational needs.



9. Review **Domains**


10. Review **Styles**


11. Review **Characteristics**


12. Click **Metadata Management**


13. Click **Connectivity** and Review 


14. Click **Integration Use Cases** and Review different use cases



15. Click **Monitoring and Operations** and review



16. Click **Transformations** to review
